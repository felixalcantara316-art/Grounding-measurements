A procedure for the determination of the resistance of grounding systems when using the fall-of-potential method, is presented. The resulting plot of the measured points are graphed in a paper from which the exact point of the resistance must be deduced. For doing this a method of least-squares for adjusting a third- or higher-degree polynomial curve is normally used.  This kind of procedure is a pure mathematical adjustment useful in any situation but ignores the physical conditions of this particular situation. equivalent hemisphere.
